# postIt
Simple pseudo CRUD app that allows the user to create, read, update, and delete posts. NO JS USED.

## Features
<b>User Allowance: </b>

 - [Create Posts]
 - [Read Posts]
 - [Updated Post (User Only)]
 - [Delete Post (User Only)]

 - [Login]
 - [Register]
 - [Reset Password]
